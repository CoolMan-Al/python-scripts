# RGB LED <br/>
## Python Scripts with an RGB LED on a breadboard
  - Scripts:
    - Rainbow Flow Light (Stoppable with button)
    - Reaction Game (2 Player capability)

  - Parts needed:
    - Breadboard
    - RGB LED
    - 3x Resistor
    - 2x Buttons
    - 6x F-M Jumper Wires
    - 3x M-M Jumper Wires
