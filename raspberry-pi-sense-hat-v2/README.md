# Sense HAT V2 <br/>
## Python Scripts with the Sense HAT
### Scripts:

#### Sensor Readings
Reads the onboard sensors on the Sense HAT and outputs them to the LED Matrix

#### Binary Clock
Pulls the Pi's system clock, converts the hours, minutes and seconds to binary bytes and outputs them onto the LED Matrix

#### Dice
Uses the Sense HAT's motion sensor to detect shaking, then outputs a randomised number between 1 and 6/20.
